Certainly. You can use the built-in sorted() function or the list.sort() method with a lambda function as the key.

terminal Implementation
main.py 
data = [
    {'name': 'Alice', 'age': 30},
    {'name': 'Bob', 'age': 25},
    {'name': 'Charlie', 'age': 35}
]
# Sort by 'age' using lambda
sorted_data = sorted(data, key=lambda x: x['age'])
print(sorted_data)

info Key Takeaways

  • The sorted() function returns a new list.
  • Original data remains unchanged unless assigned back.
  • Sorting is stable and uses Timsort algorithm.
