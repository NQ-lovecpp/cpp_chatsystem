Write a Python script to sort a list of dictionaries by a specific key.

Certainly. You can use the built-in sorted() function or the list.sort() method with a lambda function as the key.

python 
data = [
    {'name': 'Alice', 'age': 30},
    {'name': 'Bob', 'age': 25},
    {'name': 'Charlie', 'age': 35}
]
# Sort by 'age'
sorted_data = sorted(data, key=lambda x: x['age'])
print(sorted_data)
